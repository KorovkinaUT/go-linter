# LogLint

**LogLint** — линтер для Go, который проверяет текстовые сообщения в логах.

Поддерживаемые логгеры:

* `log`
* `log/slog`
* `zap`

Линтер можно использовать как:

* самостоятельный CLI-инструмент
* плагин для golangci-lint

Линтер проверяет следующие правила:

* лог-сообщение должно начинаться со строчной буквы
* сообщение должно содержать только английские буквы
* сообщение не должно содержать спецсимволы или эмодзи
* сообщение не должно содержать потенциально чувствительные данные (проверка через поиск ключевых слов)

## Использование

Перед использованием нужно установить необходимые зависимоти:
```
go mod tidy
``` 

Сначала нужно собрать бинарники линтера и плагина `golangci-lint`, введите:

```
make build
```

После сборки будут доступны бинарники:

```
./bin/loglint
./custom-gcl
```

После сборки бинарников, чтобы запустить линтер на каком-то пакете, введите:

```
make run PACKAGE=path/to/package
```

По умолчанию `PACKAGE=./...`, то есть линтер запускается на всём проекте.

Пример запуска:

```
make run PACKAGE=./internal/...
```

Чтобы запустить линтер как плагин для `golangci-lint`, введите:

```
make plugin PACKAGE=path/to/package
```

Также линтер и плагин можно запускать напрямую через бинарники, например:
```
./bin/loglint ./...
```

```
./custom-gcl run ./...
```

Чтобы запустить тесты:

```
make test
```

Чтобы удалить собранные бинарники:

```
make clean
```
